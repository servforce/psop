# 设备维保 Domain Pack

常见术语：
- 故障现象、故障模式、根因、备件、工器具、停机窗口、维修步骤、复测、交付、维保记录。

常见工作流模式：
- 识别设备、故障现象、影响范围和当前工况。
- 做初步故障诊断，区分已知事实、可能原因和待确认信息。
- 准备维修方案，包括安全隔离、备件、工器具和人员协同。
- 指导维修执行或生成操作检查清单。
- 完成复测验证，确认故障是否消除。
- 输出交付记录、遗留风险和后续观察建议。

质量与安全要求：
- 高风险操作必须要求现场人员遵循企业安全规程，不能让模型替代现场负责人授权。
- 对不确定根因要保留诊断分支，不要强行给唯一结论。
- 维修后必须包含复测或验证步骤，不能只停留在维修建议。
