# 所属行业

属于航空航天和宇航电子制造行业，主要面向宇航电子产品中印制板组装件的生产、装联、焊接、检验和质量控制过程。

# 使用场景

该 PDF 适用于宇航电子产品印制板组装件的人员资质确认、工具设备准备、材料选用、环境控制、静电防护、多余物控制、焊接装联、修复改装和质量检查等场景。

当需要判断 PCB 组装过程中的工艺条件、工具设备、材料、环境、焊接质量和检验要求是否符合宇航电子产品装联要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供宇航电子产品印制板组装件的通用工艺技术要求，用来指导装配准备、焊接装联、过程控制、质量检验和工艺验证。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成装联检查清单、焊接工艺核对表、质量检查记录、问题整改建议或工艺验证资料。
