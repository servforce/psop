# GB/T 38342-2019《宇航电子产品 印制板组装件组装要求》参考索引

## 1. 标准定位

本文件是 [GB/T 38342-2019《宇航电子产品 印制板组装件组装要求》](./宇航电子产品%20印制板组装件组装要求.pdf) 的配套参考索引，用于在 [agent.md](./agent.md) 召回该标准后，帮助 skill 判断该标准是否适用于当前任务，并定位到 PDF 原文中的相关章节。

本索引文件只用于定位 PDF 原文，不承载标准条文。所有标准依据、具体要求、数值限值、表格内容、流程顺序、判定条件和标准原文措辞，均应以 [PDF 原文](./宇航电子产品%20印制板组装件组装要求.pdf) 为准。

截至 2026-06-23，本标准状态为**已发布，已实施**；实施日期为 **2020-07-01**。

## 2. 与 agent.md 的对应关系

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 宇航电子产品、印制板组装件、PCB/PCBA 装联、焊接装联、静电防护、多余物控制、修复改装和宇航电子装配质量。

当用户任务包含上述对象、系统、工艺、检测方法或安全控制要求时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本标准规定了宇航电子产品印制板组装件组装过程中有关人员、设备、工具、材料、环境及装联的通用工艺技术要求；适用于宇航电子产品印制板组装件的组装生产，其他电子产品的印制板组装件可参考执行。 具体范围见 [PDF 第 1 章“范围”](./宇航电子产品%20印制板组装件组装要求.pdf#page=4)。

本标准主要适用于以下对象或任务：

- 宇航电子产品印制板组装件的组装生产。
- 印制板组装件装联过程中的人员、工具设备、材料、环境、静电防护、多余物控制和组装要求定位。
- 印制板组装件修复改装、质量控制和焊接工艺验证的章节定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及印制板组装件组装生产的电子产品设计、原理图设计或 PCB 裸板制造任务。
- 仅涉及电子产品整机功能测试、可靠性试验或软件验证，而不涉及印制板组装件组装过程的任务。
- 普通电子产品 PCBA 任务若未要求参照宇航电子产品组装要求，不应仅凭本标准作为主要依据。

本节依据 [PDF 第 1 章“范围”](./宇航电子产品%20印制板组装件组装要求.pdf#page=4) 确定。超出该范围的任务，不应仅凭本标准处理。
