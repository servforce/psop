# 使用场景

该 PDF 适用于电解去毛刺机床的型式和参数确认、几何精度检测、工作精度检测、检测数据记录、设备验收和精度检验报告整理等场景。

当需要判断机床工作台平面度、Z 轴垂直度、阴极安装板平行度、工件去毛刺效果和倒圆角是否满足精度要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供电解去毛刺机床精度检验的技术要求和检测依据，用来指导检测项目确认、公差核对、测量方法选择和结果判定。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成精度检验清单、检测记录表、公差核对表、验收报告或整改建议。
