## 3. 适用范围

本文件规定了电解去毛刺机床的型式和参数、几何精度和工作精度，以及与上述精度检测相对应的公差值；适用于电解去毛刺机床精度的检测，不适用于机床运行试验和性能参数检查。 具体范围见 [PDF 第 1 章“范围”](./电解去毛刺机床%20精度检验.pdf#page=7)。

本标准主要适用于以下对象或任务：

- 电解去毛刺机床精度检测。
- 机床型式、主要参数、几何精度和工作精度定位。
- 工作台台面平面度、Z 轴运动垂直度、工作台台面与阴极安装板平行度、交叉孔去毛刺效果和倒圆角检查定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 机床运行试验，例如振动、异常噪声、零部件爬行等检查。
- 机床性能参数检查，例如速度、进给量等检查。
- 不涉及电解去毛刺机床精度检测的普通机床加工或去毛刺工艺任务。

本节依据 [PDF 第 1 章“范围”](./电解去毛刺机床%20精度检验.pdf#page=7) 确定。超出该范围的任务，不应仅凭本标准处理。
