## 3. 适用范围

本文件规定了铸件采用存储磷光成像板（IP）的工业计算机 X 射线和 γ 射线照相检测技术分级、一般要求、检测技术、图像评定、检测记录和报告；适用于铸钢、铸铁、铜及铜合金、镍及镍合金、镁及镁合金、铝及铝合金、钛及钛合金等材料铸件的工业计算机射线照相检测，其他铸件参照使用。 具体范围见 [PDF 第 1 章“范围”](./铸件%20工业计算机射线照相检测.pdf#page=7)。

本标准主要适用于以下对象或任务：

- 铸件采用存储磷光成像板（IP）的工业计算机射线照相检测。
- 工业计算机 X 射线和 γ 射线照相检测技术分级、一般要求、检测技术和图像评定。
- 铸件 CR 检测记录、检测报告、归一化信噪比、图像最低像质值、最小灰度值、基本空间分辨率和环形铸件最少透照次数定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及铸件工业计算机射线照相检测的普通铸造工艺、热处理、机加工或外观检验任务。
- 胶片射线照相检测、超声检测、磁粉检测、渗透检测等其他无损检测方法任务，不应仅凭本标准作为主要依据。
- 需要缺陷验收等级、具体铸件图纸判定或合同专用验收准则的任务。

本节依据 [PDF 第 1 章“范围”](./铸件%20工业计算机射线照相检测.pdf#page=7) 确定。超出该范围的任务，不应仅凭本标准处理。
