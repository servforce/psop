# 所属行业

属于铸造和无损检测行业，主要面向铸钢、铸铁、有色金属及合金铸件的工业计算机射线照相检测过程。

# 使用场景

该 PDF 适用于铸件工业计算机射线照相检测的订货要求确认、检测技术等级选择、检测工艺文件编制、透照参数核对、图像质量评定、检测记录和检测报告编制等场景。

当需要判断铸件 CR 射线照相检测的人员、设备、工艺、图像质量、评定记录和报告内容是否符合要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供铸件工业计算机射线照相检测的技术要求，用来指导检测准备、工艺编制、透照实施、图像评定、记录填写和报告整理。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成检测工艺核对表、图像质量检查清单、检测记录模板、报告整理建议或问题整改建议。
