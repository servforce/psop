# 国家/行业标准资料库入口

## 1. 使用目的

本文件是 `standard` 标准资料库的总入口，用于帮助 skill 快速找到可能相关的国家标准或行业标准。

本文件只负责候选标准召回，不负责最终适用性判断。命中候选标准后，应继续读取对应的索引文件；涉及具体数值、表格、公式、图示和标准原文措辞时，应回到对应 PDF 原文。

## 2. 检索流程

1. 优先匹配用户明确提到的标准编号或标准名称。
2. 若没有明确编号或名称，根据 [标准目录](#3-标准目录) 中各标准的“候选召回提示”初筛候选标准。
3. 命中候选标准后，打开对应索引文件，确认该标准是否真正适用于当前任务。
4. 若索引文件确认相关，再按其中索引定位 PDF 的章节、表、图或附录。
5. 若问题涉及具体数值、公式、表格、图示、判定条件或标准原文措辞，必须打开对应 PDF 原文查证。
6. 若本文件无法判断，应遍历各索引文件的头部信息、适用范围和任务匹配部分，而不是直接读取所有 PDF。
7. 不应仅依据“候选召回提示”生成最终标准结论；候选召回提示只用于初筛。

## 4. 候选召回提示来源和写法

候选召回提示用于帮助 skill 在没有明确标准编号或标准名称时，从标准目录中召回可能相关的候选标准。

编写候选召回提示时，应按以下来源优先级取词：

1. **标准编号和标准名称**：优先使用 PDF 首页或标准公开来源中的正式名称。
2. **标准范围**：使用 PDF 第 1 章“范围”中明确出现的对象、工艺、设备、材料或方法。
3. **PDF 目录标题**：使用一级章节、重要二级章节、附录、表或图标题中的任务词。
4. **对应索引文件的使用场景**：使用已人工确认过的应用场景、检查对象和输出类型。
5. **行业常用简称**：只加入非常常见且不会造成误判的简称，例如 `SIS`、`CR 检测`、`PCB/PCBA`。

候选召回提示应优先包含强区分词：标准对象、专用系统、专用工艺、专用检测方法、专用技术参数或行业简称。

候选召回提示不应包含以下内容：

- PDF 中没有依据的扩展场景。
- 具体技术数值、公式、限值或验收结论。
- 与标准范围关系很弱的泛泛行业词。
- 会让该标准误匹配到其他领域的宽泛词。
- 将“报告、记录、清单、检查、整改、验收”等泛化输出词作为主要匹配依据；若确需出现，应与强区分对象绑定，例如“铸件 CR 检测报告”而不是单独写“检测报告”。

候选召回提示只用于初筛。命中后必须读取对应索引文件，由索引文件判断标准是否真正适用于当前任务。

## 5. 匹配规则

1. **标准编号精确匹配优先**：如果用户明确提到标准编号，应直接使用对应标准作为候选。
2. **标准名称精确匹配其次**：如果用户明确提到标准名称或接近完整的标准标题，应使用对应标准作为候选。
3. **候选召回提示只做初筛**：如果用户没有给出明确标准编号或名称，可根据候选召回提示初筛候选，但不能仅凭候选召回提示判定最终适用。
4. **候选标准需要二次确认**：命中候选后，应读取对应索引文件，由索引文件的适用范围、任务映射和使用边界确认是否真正相关。
5. **无法判断时扩大读取范围**：若 `agent.md` 无法选出候选，可依次读取各索引文件的头部信息、适用范围和任务匹配部分，不应直接读取所有 PDF。

## 6. 命中标准后的使用方式

命中候选标准后，应按以下顺序使用资料：

1. 打开该标准的索引文件，确认标准定位、适用范围、任务场景和章节索引。
2. 根据索引文件中的链接定位 PDF 的章节、表、图或附录。
3. 需要准确数值、公式、表格、图示、判定条件或标准原文措辞时，打开 PDF 原文查证。
4. 输出时不要把索引文件当作标准原文；索引文件只是定位说明，PDF 才是最终依据。

## 8. 使用边界

- 本文件只负责候选标准召回和资料定位，不替代任何标准原文。
- 对标准条文、技术数值、检验项目、公式、表格和图示的最终判断，应以对应 PDF 原文为准。
- 标准适用性还可能受合同、图纸、技术条件、客户要求、验收标准、法律法规、现场安全条件和企业制度影响。
- 如果用户任务不属于本目录中任何标准的适用范围，应说明未找到明确匹配标准，而不是强行套用相近标准。

## 9. 维护规则

新增标准时，应在 [标准目录](#3-标准目录) 中新增一个标准卡片，并保证：

- 标题使用标准编号和标准名称，例如 `### GB/T 44921-2024《铸件 工业计算机射线照相检测》`。
- 索引文件链接指向对应 Markdown 索引文件。
- PDF 原文链接指向对应 PDF 文件。
- 状态应区分“已实施”和“尚未实施”，并与实施日期一致。
- 候选召回提示只写一句用于初筛的强区分提示，详细适用范围和任务映射应写入对应索引文件。
