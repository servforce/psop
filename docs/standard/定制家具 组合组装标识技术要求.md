# GB/T 39019-2020《定制家具 组合组装标识技术要求》参考索引

本文件是 [GB/T 39019-2020《定制家具 组合组装标识技术要求》](./定制家具%20组合组装标识技术要求.pdf) 的配套参考索引，用于在 [agent.md](./agent.md) 召回该标准后，帮助 skill 判断该标准是否适用于当前任务，并定位到 PDF 原文中的相关章节。

本索引文件只用于定位 PDF 原文，不承载标准条文。所有标准依据、具体要求、数值限值、表格内容、流程顺序、判定条件和标准原文措辞，均应以 [PDF 原文](./定制家具%20组合组装标识技术要求.pdf) 为准。

截至 2026-06-23，本标准状态为**已发布，已实施**；实施日期为 **2021-02-01**。

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 定制家具、组合组装标识、家具部件标识、包装件标识、板式家具、现场组装标识、安装标识和售后追溯标识。

当用户任务包含上述对象、标识类型、组装过程或追溯场景时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本标准规定了定制家具组合组装标识的术语和定义、要求。本标准适用于定制家具组合组装标识。具体范围见 [PDF 第 1 章“范围”](./定制家具%20组合组装标识技术要求.pdf#page=3)。

本标准主要适用于以下对象或任务：

- 定制家具组合组装标识相关任务。
- 定制家具零部件标识、组合组装过程标识和成品标识相关任务。
- 标识内容、标识位置、电子标识形式、文字/符号要求和相关示例的定位任务。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及定制家具组合组装标识的家具设计、制造、安装或售后任务。
- 仅涉及家具结构强度、安全性能、环保性能、外观质量或产品检测，而不涉及组合组装标识的任务。
- 普通物流标签、仓储标签、企业内部编码规则或非定制家具标识任务。
- 需要完整安装作业指导书、具体企业编码系统、二维码数据结构或信息系统接口规则的任务。

本节依据 [PDF 第 1 章“范围”](./定制家具%20组合组装标识技术要求.pdf#page=3) 确定。超出该范围的任务，不应仅凭本标准处理。
