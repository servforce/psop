# 所属行业

属于家具制造行业，主要面向定制家具、板式家具、组合式家具等产品的生产、包装、安装和交付过程。

# 使用场景

该 PDF 适用于定制家具在生产加工、部件标识、包装发运、现场组装、安装验收和售后追溯等场景。

当需要判断家具部件、包装件、组合件的标识是否清晰、完整、便于装配和追溯时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供定制家具组合组装标识的技术要求，用来指导如何设置标识、检查标识是否符合要求，以及判断标识是否能支持后续的包装、安装和追溯。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成检查清单、安装核对表、验收记录或整改建议。
