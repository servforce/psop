# 所属行业

属于机械制造和锻压机械检测装备行业，主要面向机械压力机、液压机用模拟负荷测试系统的制造、试验、检验和质量验收过程。

# 使用场景

该 PDF 适用于模拟负荷测试系统的技术要求核对、机械性能试验、精度和功能检查、性能试验、安全性试验、出厂检验、型式检验、标志包装和贮运要求检查等场景。

当需要判断测试系统的过载能力、耐热性、基本误差、重复性误差、响应时间、绝缘电阻、绝缘强度和检验项目是否符合要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供机械压力机、液压机用模拟负荷测试系统的技术要求、试验方法和检验规则，用来指导产品试验、检验和验收判断。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成试验检查清单、出厂检验记录、型式检验核对表、验收报告或整改建议。
