# GB/T 21681-2025《机械压力机、液压机用模拟负荷测试系统》参考索引

## 1. 标准定位

本文件是 [GB/T 21681-2025《机械压力机、液压机用模拟负荷测试系统》](./机械压力机、液压机用模拟负荷测试系统.pdf) 的配套参考索引，用于在 [agent.md](./agent.md) 召回该标准后，帮助 skill 判断该标准是否适用于当前任务，并定位到 PDF 原文中的相关章节。

本索引文件只用于定位 PDF 原文，不承载标准条文。所有标准依据、具体要求、数值限值、表格内容、流程顺序、判定条件和标准原文措辞，均应以 [PDF 原文](./机械压力机、液压机用模拟负荷测试系统.pdf) 为准。

截至 2026-06-23，本标准状态为**已发布，已实施**；实施日期为 **2026-05-01**。

## 2. 与 agent.md 的对应关系

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 机械压力机、液压机、模拟负荷测试系统、荷重传感器、极限过载、安全过载、基本误差、重复性误差、响应时间、绝缘电阻和绝缘强度。

当用户任务包含上述对象、系统、工艺、检测方法或安全控制要求时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本文件规定了机械压力机、液压机用模拟负荷测试系统的技术要求，描述了相应的试验方法，规定了检验规则、标志、包装、贮存和运输的内容；适用于机械压力机、液压机用模拟负荷测试系统的制造。 具体范围见 [PDF 第 1 章“范围”](./机械压力机、液压机用模拟负荷测试系统.pdf#page=7)。

本标准主要适用于以下对象或任务：

- 机械压力机、液压机用模拟负荷测试系统的制造。
- 模拟负荷测试系统技术要求、试验方法、检验规则和标志包装贮运定位。
- 荷重传感器、极限过载、安全过载、精度、功能、性能和安全性要求定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及机械压力机、液压机用模拟负荷测试系统制造的压力机或液压机本体设计任务。
- 仅涉及压力机、液压机生产工艺、模具设计或设备运行维护，而不涉及模拟负荷测试系统的任务。
- 需要具体厂家测试系统说明书、校准证书或企业内部检验规程的任务。

本节依据 [PDF 第 1 章“范围”](./机械压力机、液压机用模拟负荷测试系统.pdf#page=7) 确定。超出该范围的任务，不应仅凭本标准处理。
