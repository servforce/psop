# 所属行业

属于新能源汽车维修和汽车后市场服务行业，主要面向纯电动汽车、插电式混合动力汽车以及使用气态氢的燃料电池汽车维修作业。

# 使用场景

该 PDF 适用于新能源汽车维修中的人员资质确认、维修场地检查、作业准备、风险排查与处置、高压系统维修、动力蓄电池维修、车载供氢系统维修和操作安全检查等场景。

当需要判断新能源汽车维修作业中的人员、场地、防护装备、风险控制和操作安全要求是否满足标准要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供新能源汽车维修作业安全要求，用来指导维修前准备、现场安全检查、风险排查、作业流程控制和操作安全核对。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成安全检查清单、作业准备核对表、风险排查记录、维修安全提示或整改建议。
