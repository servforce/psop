# GB/T 47439-2026《新能源汽车维修作业安全要求》参考索引

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 新能源汽车维修、高压系统维修、动力蓄电池维修、车载供氢系统维修、涉电作业、涉氢作业、绝缘防护和维修作业安全。

当用户任务包含上述对象、系统、工艺、检测方法或安全控制要求时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本标准规定新能源汽车维修作业中的人员与场地、作业准备、风险排查与处置、作业流程和操作安全要求。 具体范围见 [PDF 第 1 章“范围”](./新能源汽车维修作业安全要求.pdf#page=7)。

本标准主要适用于以下对象或任务：

- 纯电动汽车。
- 插电式混合动力电动汽车。
- 使用气态氢的燃料电池电动汽车。
- 高压系统及相关附件维修。
- 动力蓄电池相关维修。
- 车载供氢系统和燃料电池系统相关维修。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及新能源汽车专用装置维修作业的普通车辆维修任务。
- 仅涉及整车设计、制造、产品认证或型式试验的任务。
- 需要具体车型拆装步骤、诊断流程、故障码处理或专用工具操作方法的任务。

本节依据 [PDF 第 1 章“范围”](./新能源汽车维修作业安全要求.pdf#page=7) 确定。超出该范围的任务，不应仅凭本标准处理。
