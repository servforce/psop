# GB/T 47696-2026《过程工业安全仪表系统检验测试》参考索引

## 1. 标准定位

本文件是 [GB/T 47696-2026《过程工业安全仪表系统检验测试》](./过程工业安全仪表系统检验测试.pdf) 的配套参考索引，用于在 [agent.md](./agent.md) 召回该标准后，帮助 skill 判断该标准是否适用于当前任务，并定位到 PDF 原文中的相关章节。

本索引文件只用于定位 PDF 原文，不承载标准条文。所有标准依据、具体要求、数值限值、表格内容、流程顺序、判定条件和标准原文措辞，均应以 [PDF 原文](./过程工业安全仪表系统检验测试.pdf) 为准。

截至 2026-06-23，本标准状态为**已发布，尚未实施**；实施日期为 **2026-12-01**。引用该标准时，应明确说明其尚未实施状态，不应表述为已经实施的现行执行依据。

## 2. 与 agent.md 的对应关系

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 过程工业安全仪表系统、SIS、proof test、功能安全、检验测试规划、检验测试覆盖率、安全仪表功能、测试旁路和运行维护数据。

当用户任务包含上述对象、系统、工艺、检测方法或安全控制要求时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本文件规定了过程工业安全仪表系统检验测试的总则、规划、实施、分析、报告和整改的要求；适用于过程工业领域安全仪表系统的检验测试。 具体范围见 [PDF 第 1 章“范围”](./过程工业安全仪表系统检验测试.pdf#page=7)。

本标准主要适用于以下对象或任务：

- 过程工业领域安全仪表系统（SIS）的检验测试。
- SIF、SIL、PTC、PFDavg/PFH、端到端/分段/单体设备检验测试相关任务定位。
- 检验测试规划、实施、分析、报告和整改相关任务定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及过程工业安全仪表系统检验测试的普通仪表校验、常规控制系统维护或电气维修任务。
- 仅涉及 SIS 设计、安装调试、验收或 SIL 确定，而不涉及运行期间检验测试的任务。
- 需要具体企业 SRS、现场联锁逻辑、设备安全手册或厂家测试程序的任务。

本节依据 [PDF 第 1 章“范围”](./过程工业安全仪表系统检验测试.pdf#page=7) 确定。超出该范围的任务，不应仅凭本标准处理。
