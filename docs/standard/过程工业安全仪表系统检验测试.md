# 所属行业

属于过程工业、工业自动化和功能安全行业，主要面向石油、化工、天然气、电力等过程工业现场的安全仪表系统运行维护过程。

# 使用场景

该 PDF 适用于安全仪表系统 SIS 的检验测试规划、测试准备、测试执行、测试分析、测试报告、问题整改和运行维护数据记录等场景。

当需要判断 SIS 检验测试内容、测试方式、测试流程、覆盖率分析、报告记录和整改要求是否符合功能安全要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供过程工业安全仪表系统检验测试的要求，用来指导检验测试规划、实施、分析、报告和整改工作。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成 SIS 检验测试计划、测试检查清单、测试记录表、报告模板、分析说明或整改建议。
