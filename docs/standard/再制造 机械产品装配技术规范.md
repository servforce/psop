# GB/T 40727-2021《再制造 机械产品装配技术规范》参考索引

## 1. 标准定位

本文件是 [GB/T 40727-2021《再制造 机械产品装配技术规范》](./再制造%20机械产品装配技术规范.pdf) 的配套参考索引，用于在 [agent.md](./agent.md) 召回该标准后，帮助 skill 判断该标准是否适用于当前任务，并定位到 PDF 原文中的相关章节。

本索引文件只用于定位 PDF 原文，不承载标准条文。所有标准依据、具体要求、数值限值、表格内容、流程顺序、判定条件和标准原文措辞，均应以 [PDF 原文](./再制造%20机械产品装配技术规范.pdf) 为准。

截至 2026-06-23，本标准状态为**已发布，已实施**；实施日期为 **2022-05-01**。

## 2. 与 agent.md 的对应关系

本标准在 [agent.md](./agent.md) 中的候选召回提示为：

> 再制造机械产品、机械产品装配、零部件装配、总成装配、压装、热装、冷装、密封件装配、装配质量检验和信息追溯。

当用户任务包含上述对象、系统、工艺、检测方法或安全控制要求时，可将本标准作为候选标准；但是否真正适用，应继续根据本文件的 [适用范围](#3-适用范围)、[不适用范围](#4-不适用范围) 和 [任务到章节映射](#5-任务到章节映射) 确认。

## 3. 适用范围

本文件规定了再制造机械产品装配的一般要求、再制造装配过程、主要装配方式，以及再制造产品装配质量检验、标识和信息管理、安全与环保要求等；适用于再制造机械产品的装配过程，其他再制造产品装配也可参照执行。 具体范围见 [PDF 第 1 章“范围”](./再制造%20机械产品装配技术规范.pdf#page=3)。

本标准主要适用于以下对象或任务：

- 再制造机械产品的装配过程。
- 再制造零部件、总成或产品装配过程中的工艺选择、装配方式和质量检验定位。
- 再制造产品装配后的标识、信息溯源、安全与环保要求定位。

## 4. 不适用范围

以下任务不应将本标准作为主要依据：

- 不涉及再制造机械产品装配过程的普通制造、维修或使用任务。
- 仅涉及再制造前的拆解、清洗、检测、修复加工，而不涉及装配过程的任务。
- 需要具体产品型号的装配作业指导书、专用工装参数或企业内部工艺卡的任务。

本节依据 [PDF 第 1 章“范围”](./再制造%20机械产品装配技术规范.pdf#page=3) 确定。超出该范围的任务，不应仅凭本标准处理。
