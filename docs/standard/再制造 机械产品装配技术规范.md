# 所属行业

属于机械制造和再制造行业，主要面向再制造机械产品的零部件装配、总成装配、质量检验和过程管理。

# 使用场景

该 PDF 适用于再制造机械产品装配前检查、装配工艺选择、压装、热装、冷装、密封件装配、装配质量检验、标识管理、信息追溯和安全环保检查等场景。

当需要判断再制造机械产品的装配过程、装配方式、质量检验和信息管理是否满足技术要求时，可以参考该 PDF。

# 素材定位

该 PDF 是一个标准类指导文件，不是一个完整的现场操作步骤手册。

它的主要作用是提供再制造机械产品装配过程中的技术规范，用来指导装配方法选择、装配过程控制、装配质量检验、产品标识和信息追溯。

在 PSOP 中，该 PDF 可以作为 PSkill 执行过程中的参考素材，用于生成装配检查清单、工艺选择建议、质量检验记录、安全环保核对表或整改建议。
