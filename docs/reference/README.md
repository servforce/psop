# Reference Docs

- API 字段字典
- WebSocket 事件字典
- 配置项字典
- 排障手册
- 运行运维手册

- 当前版本暂不单独拆出 API 或数据库字典。
- 接口与数据库细节暂时内聚在 [../PSOP服务端详细设计v1.md](../PSOP%E6%9C%8D%E5%8A%A1%E7%AB%AF%E8%AF%A6%E7%BB%86%E8%AE%BE%E8%AE%A1v1.md) 中，以便尽快启动开发。

## 维护原则

- 只有当接口与表结构进入稳定期后，才从服务端详细设计中抽取到本目录。
- 本目录只收稳定契约，不收大段架构推导或方案比较。
