# UI Docs

本目录保留给未来的界面专题补充文档，例如：

- 页面原型与交互草图
- 组件设计补充说明
- 可用性测试记录
- 视觉规范补充材料

## 当前阶段说明

- 当前有效的前端详细设计基线是 [../PSOP前端详细设计v1.md](../PSOP%E5%89%8D%E7%AB%AF%E8%AF%A6%E7%BB%86%E8%AE%BE%E8%AE%A1v1.md)。
- 本目录暂不承载一线实现基线，后续新增文档只能作为补充说明。

## 维护原则

- 根目录详细设计负责回答“系统应该如何实现”。
- 本目录文档负责回答“某个页面或交互专题如何展开”。
- 如本目录文档与根目录详细设计冲突，以根目录详细设计为准。
