# Research

本目录保存外部调研、技术预研、方案比较和阶段性建议。调研文档可以影响架构决策，但不能直接替代架构事实源。

- [run-server-runtime-research.md](run-server-runtime-research.md)
  - Run Server 相关外部项目调研与运行时架构建议。
- [ahpu-edge-inference-research.md](ahpu-edge-inference-research.md)
  - AHPU、端侧推理、模型优化、硬件化可行性预研。

## 维护原则

- 调研文档应标注资料范围、检索基线日期和结论适用边界。
- 调研结论如果被采纳，应沉淀到 `../architecture/` 或 `../overview/` 的正式文档中。
- 过期调研如仍有历史价值，保留在本目录或移入 `../archive/`，但必须标注状态。
