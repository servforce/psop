# Overview

本目录保存产品愿景、产品规划、路线图、术语和高层产品原则。这里的文档回答“为什么做”和“要走向哪里”，不直接替代架构基线。

## 当前文档

- [vision.md](vision.md)
  - 产品判断、业务闭环、智能体定位、阶段目标、成功标准与术语。
- [product-safety-plan-v1.md](product-safety-plan-v1.md)
  - 产品安全主张、安全旅程、安全能力地图与阶段性路线图。

## 维护原则

- 产品方向变化先更新本目录，再同步检查 `../architecture/` 是否需要调整。
- 本目录可以提出目标能力和路线图，但不能把尚未落地的实现写成当前事实。
- 术语变更应同步影响 `vision.md` 和架构文档中的核心对象定义。
