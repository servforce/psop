# References

本目录用于存放 `skill-compiler` 编译某个 Skill 时可参考的规范资料。

当前阶段建议包含的参考资料类型：

- Skill 最小目录结构标准
- `skill.yaml` 字段说明
- 最小 runtime bundle 结构说明
- 构建报告格式说明

未来如需增强，可继续增加：

- `skill-directory-standard.md`
- `skill-yaml-contract.md`
- `runtime-bundle-format.md`
- `build-report-format.md`

当前阶段该目录主要作为标准说明入口存在，不要求包含复杂内容。
