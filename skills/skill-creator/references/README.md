# References

本目录用于放置 `skill-creator` 在生成目标 Skill 草稿时需要引用的参考资料。

当前阶段建议引用的资料类型包括：

- Skill 目录结构标准
- `SKILL.md` 编写规范
- `skill.yaml` 字段规范
- 示例 Skill 样本
- 验收清单模板

如果后续需要扩展，可在此目录中继续加入：

- `skill-directory-standard.md`
- `skill-yaml-schema.md`
- `example-skills.md`
- `review-guidelines.md`

当前文件作为目录占位与用途说明，用于提醒后续维护者：`references/` 是正式标准目录名，不使用 `knowledge/`。
